# Dashboard Web v2.4.0

Dashboard web para consultar a API Central e visualizar clientes/Agents, status, alertas e últimas métricas.

## Rodar

```powershell
cd dashboard-web
npm install
npm run dev
```

Acesse:

```txt
http://localhost:5174
```

Use:

```txt
API: http://127.0.0.1:4090
Token: dev-token-change-me
```
